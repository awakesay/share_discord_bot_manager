概要
    ローカルマシンのDiscord bot管理、及びローカルマシンのコマンドをbot経由で実行するDiscord botです。

事前準備
    Discord botのAPIキー（/config/discord_bot.json)
    管理するDiscord botの起動設定（/config/bots.json）
    ローカルマシンの実行権限設定（/config/do_cmd_permission.json）

動作環境
    PC: Raspberry Pi 4
    OS: Raspberry pi OS

言語・パッケージ
    Python 3.9.2
    pycord
    tabulate

サービス
    なし
